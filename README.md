# Haste Framework
`Haste Framework`는 UDP(User Datagram Protocol)를 기반으로 빠르고 쉬운 개발을 위해 만들어진 게임 서버 프레임워크 이다.

`Haste 란? : 속도를 빠르게 만드는 효과를 가진 가속 마법을 의미한다.`
